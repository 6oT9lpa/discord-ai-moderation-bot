# 📋 Шпаргалка - AI Module\n\n## 🚀 Быстрый старт\n\n### Terminal 1: Start Ollama\n```bash\nollama serve\n```\n\n### Terminal 2: Download Model\n```bash\nollama pull mistral\n# или быстрее: ollama pull neural-chat\n# или лучше: ollama pull llama3\n```\n\n### Terminal 3: Configure & Start Bot\n```bash\ncd discord-ai-moderation-bot\nedit .env\n# Установить: OLLAMA_ENABLED=true\npython main.py\n```\n\n---\n\n## 🎮 Discord Commands\n\n| Команда | Что делает |\n|---------|----------|\n| `/ai_chat <msg>` | 💬 Общайтесь с ИИ |\n| `/ai_clear` | 🗑️ Очистить историю |\n| `/ai_history` | 📝 Показать последние 10 сообщений |\n| `/ai_stats` | 📊 Статистика модерации |\n| `/ai_test` | 🔌 Проверить подключение |\n\n---\n\n## ⚙️ .env Configuration\n\n```env\n# Включить/отключить AI\nOLLAMA_ENABLED=true\n\n# Сервер Ollama\nOLLAMA_BASE_URL=http://localhost:11434\n\n# Модель (mistral, neural-chat, llama3, phi3:mini)\nOLLAMA_MODEL=mistral\n```\n\n---\n\n## 🛠️ Ollama Commands\n\n```bash\n# Список загруженных моделей\nollama list\n\n# Скачать модель\nollama pull mistral\n\n# Запустить модель (тестирование)\nollama run mistral\n\n# Удалить модель\nollama rm mistral\n\n# Показать информацию о модели\nollama show mistral\n```\n\n---\n\n## 📦 Создать custom Modelfile\n\n```bash\n# Создать model из Modelfile\nollama create my-bot -f Modelfile\n\n# Затем использовать в .env:\n# OLLAMA_MODEL=my-bot\n\n# Удалить\nollama rm my-bot\n```\n\n---\n\n## 🐛 Troubleshooting\n\n### \"Connection refused\"\n→ Убедитесь `ollama serve` запущен\n\n### \"Model not found\"\n→ Скачайте: `ollama pull mistral`\n\n### \"Timeout\"\n→ Проверьте ресурсы (RAM, CPU)\n\n### \"Slow responses\"\n→ Используйте быструю модель: `ollama pull neural-chat`\n\n---\n\n## 📊 Производительность\n\n| Метрика | Значение |\n|---------|----------|\n| Первый ответ | 3-10 сек |\n| Последующие | 1-5 сек |\n| Классификация | 2-3 сек |\n| Default timeout | 30 сек |\n\n---\n\n## 🎯 Model Selection\n\n### Скорость (быстрые)\n- `neural-chat` - ⚡⚡⚡ (самая быстрая)\n- `mistral` - ⚡⚡ (хороший баланс)\n\n### Качество (лучшие ответы)\n- `llama3` - ⭐⭐⭐⭐⭐ (лучшая)\n- `mistral` - ⭐⭐⭐⭐ (хорошая)\n\n### Маленькие (слабое железо)\n- `phi3:mini` - 3.8B (очень маленькая)\n\n---\n\n## 📁 Файлы\n\n| Файл | Назначение |\n|------|----------|\n| `Modelfile` | Конфиг для оптимизированной модели |\n| `.env` | Конфигурация бота |\n| `QUICK_START_AI.md` | Быстрый старт |\n| `AI_MODERATION_README.md` | Полная документация |\n| `MODELFILE_GUIDE.md` | Гайд по Modelfile |\n\n---\n\n## ✨ Что делает AI?\n\nАИ **видит:**\n- Ваш Discord ник\n- Ваши роли на сервере\n- Ваш статус (admin или user)\n- История ваших сообщений (20 последних)\n\nАИ **делает:**\n- Отвечает на вопросы с учетом контекста\n- Помнит что вы обсуждали\n- Учитывает ваш статус (даёт разные ответы администраторам)\n\n---\n\n## 📝 Пример использования\n\n```\n👤 User (admin): /ai_chat How do I setup roles?\n\n🤖 Bot: \n👤 Администратор, для установки ролей:\n1.
